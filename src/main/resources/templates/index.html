String Analyzer API

A RESTful service for analyzing strings and storing their computed properties.

Overview

This API allows you to submit strings for analysis and retrieve computed properties such as:

  • Length
  • Palindrome check
  • Unique characters
  • Word count
  • SHA-256 hash
  • Character frequency map

Available Endpoints

  • POST /strings – Analyze and store a new string
  • GET /strings/{string_value} – Retrieve analysis for a specific string
  • GET /strings – Get all strings (with optional filters)
  • GET /strings/filter-by-natural-language – Use natural language to filter results
  • DELETE /strings/{string_value} – Remove a string from the database

Example Request

POST /strings

{
  "value": "Hello world"
}

Response:

{
  "id": "64ec88ca00b268e5...",
  "value": "Hello world",
  "properties": {
    "length": 11,
    "is_palindrome": false,
    "unique_characters": 8,
    "word_count": 2,
    "sha256_hash": "64ec88ca00b268e5...",
    "character_frequency_map": {
      "H": 1, "e": 1, "l": 3, "o": 2, " ": 1, "w": 1, "r": 1, "d": 1
    }
  },
  "created_at": "2025-10-20T21:51:25.07"
}
